# 🧠 Chicorun Problem Quality Agent Reference

## 02_chicorun_planning.md (02_chicorun_planning.md)
본 문서는 치코런 서비스의 문제 생성 시스템에 대한 지침을 담고 있습니다. 특히 AI 기반 문제 생성 시 발생할 수 있는 오류를 방지하고, 교육적 가치가 높은 문제를 생성하기 위한 품질 보장 명세를 포함합니다.

---

## AI 작업 지침

### 목적
* 모든 생성 문제는 **정답이 정확히 1개**여야 하며, 교육적으로 올바른 문법과 자연스러운 의미를 가져야 함.
* 그럴듯하지만 명확히 틀린 오답(Distractors)을 생성하여 학습 효과를 극대화함.

### 작업 단계
1. **Verb 선택 및 규칙 조회**: 생성할 동사를 선택하고 해당 동사의 `VERB_RULES`를 조회합니다.
2. **호환되는 Object 선택**: 해당 동사와 함께 사용 가능한 `validObjects` 중 하나를 선택합니다.
3. **Template 기반 정답 후보 생성**: 동사별 문장 템플릿에 주어, 동사(변형), 목적어를 대입하여 정답을 생성합니다.
4. **오답 유형 결정**: 수일치, 시제, 구조, 관사 누락 중 서로 다른 3가지 오류 유형을 선택함.
5. **오답 생성**: 선택한 오류 유형을 적용하여 오답 문장을 생성함. 이때 실존하지 않는 문법 구조나 랜덤한 전치사 추가는 금지됨.
6. **최종 검증**: 
   - 정답이 1개인지 확인 (중복 제거)
   - 동사-목적어 조합이 유효한지 확인 (`verbObjectCombinationValid`)
   - 전치사 남용 여부 및 모든 문장의 구조적 유효성 확인

### 주의사항
* **동사-목적어 호환성**: `make + park`와 같이 의미상 어색한 조합은 절대 생성하지 않으며, 검증 단계에서 탈락시킴.
* **관사 관리**: 초급 단계에서는 관사(a/the) 유무가 정오답의 기준이 될 수 있으므로, 명확하게 처리함.
* **템플릿 준수**: 무작위 단어 나열이 아닌, 반드시 정의된 템플릿(`{subject} eats {food}`)을 기반으로 문장을 생성함.
