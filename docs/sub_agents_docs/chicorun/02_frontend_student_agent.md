# 치코런 학생 프론트엔드 레벨 시스템 Agent Reference

## 02_chicorun_planning.md (레벨 시스템 관련 발췌)

### 7. 🎯 레벨 결정 및 시작 로직 (Option 1)

#### 7.1 목표
레벨 테스트 기능이 없으므로, 학생이 처음 학습을 시작할 때 적절한 레벨에서 시작할 수 있도록 하고, 빠르게 자신의 실력에 맞는 레벨로 조정될 수 있게 한다.

#### 7.2 전체 흐름
1. 학생이 로그인 후 학습 메인 페이지에 진입
2. 진도가 0인 경우 시작 레벨 선택 모달 자동으로 표시
3. 학생이 선택한 레벨의 시작 진도(`progressIndex`) 설정 및 학습 시작
4. 언제든지 상단 '변경' 버튼으로 레벨 수정 및 최고 레벨 기록 초기화 가능

#### 7.3 주요 UI 구성 요소
* **헤더 파트**:
    - **나의 최고 레벨**: `achievedMaxLevel` 표시 및 '변경' 버튼 (모달 연결).
    - **포인트**: 현재 보유 포인트 실시간 갱신.
    - **난이도 스위처**: 초/중/고급 난이도 수동 선택 버튼 (추천 난이도 배지 포함).
* **학습 카드**:
    - **페널티 알림**: 현재 레벨 대비 실력이 너무 높으면 포인트 감소 경고 노출.
    - **진도 바**: 해당 레벨 내의 현재 문제 순서(1/12 등) 표시.
    - **연출**: 정답 시 폭죽 및 콤보(3, 5, 10연속) 애니메이션, 오답 시 MZ 스타일 힌트 노출.
* **랭킹 페이지**:
    - 글로벌 Top 30 사용자의 닉네임, 등수, 포인트, 레벨 리스트.

#### 7.4 레벨별 문제 구성
- **1-30레벨**: 12문제씩 (초급)
- **31-70레벨**: 15문제씩 (중급)
- **71-100레벨**: 18문제씩 (고급)

## AI 작업 지침

### 목적
학생이 직관적인 UI를 통해 자신의 레벨에 맞는 문제를 풀고, 성취감을 느낄 수 있는 학습/랭킹 화면을 구현합니다.

### 작업 단계
1. **학습 엔진 (`StudentLearnPage`)**: 
   - `fetchQuestion` 시 현재 난이도를 query로 전달.
   - 정답 시 `AnswerResult` 데이터(콤보, 클리어 여부)에 따른 분기 연출.
2. **레벨 관리 모달 (`LevelSelectModal`)**:
   - 5개 구간 옵션 제공.
   - 레벨 점프 시 경고 문구 노출.
   - `achievedMaxLevel` 리셋 기능 포함.
3. **랭킹 페이지 (`RankingPage`)**:
   - `GET /api/chicorun/ranking` 연동 및 등수별 스타일링.

### 주의사항
- **UX**: 콤보 애니메이션(3, 5, 10회)이 정답 박스와 겹치지 않게 조절하고, 시각적 피드백을 강화합니다.
- **데이터 흐름**: 레벨 변경 시 반드시 `progressIndex`를 해당 레벨의 시작점으로 업데이트해야 합니다.
- **예외 처리**: 오답 시 노출되는 `explanation`은 2~3줄의 MZ 말투를 유지합니다.
