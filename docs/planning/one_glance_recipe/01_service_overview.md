# 🍳 한눈 레시피 (One-glance Recipe) 서비스 개요

## 1. 개요
한눈 레시피는 사용자가 직접 입력한 재료와 조리 방법을 기반으로 **자동으로 고퀄리티 문장형 레시피를 생성**해주는 서비스입니다.

- **핵심 가치**: 
    - "입력은 최소한으로, 결과는 전문가 수준으로"
    - 요리는 사용자가 직접 정해두고 있음 (메뉴 추천이 아닌 **문장 생성기**)
- **목표**: UI는 단순하게, 문장 완성도는 세밀하게 구현

## 2. 디자인 및 핵심 방향성
- 감성 디자인보다 **기능 완성도** 우선
- 최소한의 입력 동작
- 기본값 자동 선택
- 로그인 없이 저장 가능
- 로그인 시 카카오톡 공유 가능
- 저장/공유 시 광고 노출 로직 포함

## 3. 서비스 정의 (최종 한 줄)
한눈 레시피는 재료와 조리 방법을 입력하면 자연스럽고 완성도 높은 문장형 레시피를 자동 생성해주는 **최소 입력 기반 레시피 템플릿 생성 서비스**입니다.
