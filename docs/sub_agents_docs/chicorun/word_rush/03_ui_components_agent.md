# Word Rush: UI Component 에이전트 작업 문서

## 목표
워드 러시 게임의 프론트엔드 UI 컴포넌트 마크업, 스타일링 및 애니메이션 연출을 구현합니다.

## 제약사항 및 규칙 (필독)
- `/docs/conventions/` 내의 코딩 스타일, 네이밍 문서 및 React 19 / Next.js 규칙을 준수합니다.
- CSS Module 사용 시 클래스명 규칙을 지켜야 합니다.
- `apps/web/app/chicorun/game/word-rush/` 내에서 주로 작업합니다.

## 상세 작업 내역

1. **HUD 영역 (상단 UI)**
   - `rankPoint` 와 `coin` 분리 컴포넌트 개발 (`[ 🏆 12,450 ] [ 💰 3,200 ]`)
   - rankPoint는 강조 (골드/노랑, 크고 굵게), coin은 보조 느낌 (초록/블루, 작게)

2. **인게임 컴포넌트**
   - 영어 설명(Hint) 표시부
   - 타이핑 입력 폼 영역 구성 및 키보드 이벤트 핸들링 (엔터 미입력, 바로 체크 등 기획 확인 필요)
   - 콤보 및 피드백 이펙트 (정답/오답, 카운트업 애니메이션) UI

3. **전투 연출 (레전드 시스템) 영역**
   - 좌(유저 펫), 우(레전드 펫) 레이아웃 배치
   - 공격 / 피격 애니메이션 (직접 CSS 애니메이션, 화면 흔들림 효과) 개발

4. **종료 및 결과 모달**
   - 획득 포인트 카운트업 (rankPoint, coin 시간차 표시)

이 문서를 바탕으로 UI Component 에이전트 작업을 진행해주십시오. 담당 에이전트는 복잡한 API 연동이나 상태 알고리즘 없이, `useWordRushGame` 등 Game Engine 에이전트가 제공하는 훅을 기반으로 화면만 렌더링하도록 작성합니다.
