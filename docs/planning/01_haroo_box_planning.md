# 📦 하루상자 (HaruBox) 기획 개요

## 1. 서비스 컨셉
하루상자는 짧은 시간 안에 가볍게 즐길 수 있는 **아기자기한 놀이·게임·기록 콘텐츠를 모아둔 웹 플랫폼**입니다.

사용자는 하루상자 안에서:
- 게임을 즐기고
- 소소한 기록을 남기고
- 결과물을 저장하거나 공유하며
하루의 작은 재미를 수집합니다.

---

## 2. 핵심 방향성
- 로그인은 **선택적**
- 로그인 없이도 대부분의 콘텐츠 체험 가능
- 로그인 시 **기록·저장·개인화** 제공

---

## 3. 로그인 설계

### 3.1 로그인 원칙
- **초기 진입 허들은 낮게**
- "로그인 안 해도 됨" (Guest Play First)
- 단, 일부 기능은 로그인 필요

---

## 4. 전체 페이지 구성

### 4.1 메인 페이지 (`/`)
**목적:** 하루상자의 분위기 전달 및 콘텐츠 탐색

*   **상단**:
    *   로고: 하루상자
    *   메뉴: 콘텐츠 | 로그인
*   **히어로 섹션**:
    *   문구: "오늘의 작은 재미를 담다"
*   **콘텐츠 카드 그리드**:
    *   TOBY - 선생님을 위한 서비스 툴 (외부 링크)
    *   미니게임 (Coming Soon)
*   **하단**:
    *   서비스 설명
    *   SEO 텍스트

### 4.2 콘텐츠 목록 (`/contents` - 예정)
**목적:** 제공 중인 모든 놀이 콘텐츠 모음 (현재 메인 페이지에서 바로 접근 가능하도록 구현)

*   카드형 UI
*   상태 표시: NEW, 인기, 준비중
*   클릭 시 각 콘텐츠 상세/플레이 페이지로 이동


### 4.3 TOBY (서비스 툴)
*   **URL**: `/toby` (내부 통합 또는 `https://toby-teacher.vercel.app`)
*   **컨셉**: 학교 선생님들을 위한 수업과 학급 관리에 유용한 서비스 툴 모음
*   **연동 방식**:
    *   메인 페이지 콘텐츠 카드에 노출 (New 배지)
    *   네비게이션 바에 TOBY 항목 추가

### 4.4 자아탐험 (심리 분석)
*   **URL**: `/htsm`
*   **컨셉**: 친구들의 익명 평가로 발견하는 나를 주제로 한 Johari Window 기반 심리 분석 콘텐츠!
*   **연동 방식**:
    *   메인 페이지 콘텐츠 카드에 노출 (Hot 배지)

### 4.5 롤링페이퍼
*   **URL**: `https://r-paper-web.haroo.site`
*   **컨셉**: 친구들과 추억을 나누는 롤링페이퍼
*   **연동 방식**:
    *   메인 페이지 콘텐츠 카드에 노출 (외부 링크로 이동)

### 4.6 마이페이지 (`/mypage`)
*   로그인 정보
*   콘텐츠 이용 통계
*   데이터 초기화
*   탈퇴

## 6. 확장 고려 사항
- 콘텐츠 추가 시 `/contents/{slug}` 또는 `/{slug}` 구조 유연하게 유지
- 모든 콘텐츠는 "하루상자" 세계관과 디자인 톤앤매너 공유
- 추후 모바일 앱(WebView 기반 등) 확장 가능 구조

## 7. 한 줄 요약
