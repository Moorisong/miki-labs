### 🎮 워드 레인 (Word Rain) 게임 기획서 (확장 포함)

---

# 🎯 1. 게임 개요

**워드 레인**은 상단에서 떨어지는 한글 뜻을 보고,

하단의 영어 선택지 중 정답을 빠르게 선택하는 **반응형 단어 게임**이다.

- 시간 경과에 따라 난이도 상승
- 모든 단어 제거 시 클리어
- 하나라도 바닥 도달 시 즉시 실패

---

# 🕹️ 2. 핵심 플레이 방식

1. 한글 뜻(ko)이 위에서 떨어짐
2. 영어 선택지 중 정답 클릭
3. 맞추면 해당 단어 제거
4. 놓치면 게임 종료

---

# 📦 3. 데이터 구조 활용

- `wid + mid` 기반 문제 생성
- ko → 문제
- en → 선택지

---

# 🧠 4. 문제 생성 로직

```
{
  wid,
  mid,
  ko,
  answers:[...]
}
```

- 정답 1 + 오답 N
- 동일 level 기반 오답 구성
- 셔플 후 제공

---

# ⚡ 5. 난이도 시스템

## 시간 기반 증가

- 단어 수 증가
- 낙하 속도 증가

```
fallSpeed += time * multiplier
spawnCount += time
```

---

# 🧱 6. 중복 방지 (강화)

## 1) 기본

```
wid + mid
```

---

## 2) 최근 wid 제한 ⭐

- 최근 N개 단어 재출제 금지

```
recentWids = [1,5,8,10]
```

- 새로운 문제 생성 시 제외

👉 **단어 반복 체감 제거 (핵심 UX)**

---

## 3) 동일 mid 연속 방지 ⭐

- 바로 이전 문제와 같은 의미 금지

```
prev.mid !== current.mid
```

👉 **같은 뜻 반복 방지**

---

# 🎯 8. 점수 & 콤보 시스템

## 기본 점수

```
score = correctCount * 10
```

---

## 콤보 시스템 ⭐

### 조건

- 연속 정답 시 콤보 증가
- 오답 or 놓침 → 콤보 초기화

---

### 계산 방식

```
combo += 1

score += basePoint * comboMultiplier
```

예시:

| 콤보 | 배수 |
| --- | --- |
| 1 | x1 |
| 2 | x1.2 |
| 3 | x1.5 |
| 5+ | x2 |

---

👉 **고수 플레이 유도 핵심 요소**

# 🎁 9. 아이템 시스템

## ⏸️ 시간 정지 아이템 ⭐

### 효과

- 일정 시간 동안 단어 낙하 정지

```
duration: 2~3초
```

---

### 획득 방식

- 랜덤 드랍
- 특정 점수 도달 시 지급

---

### 제한

- 1판 최대 사용 횟수 제한 (ex: 2회)

---

👉 **위기 탈출 장치 + 전략 요소**

# 🏁 10. 성공 보상 시스템 ⭐

## 기본 보상

| 조건 | 포인트 |
| --- | --- |
| 실패 | 0 |
| 성공 | 100 |

---

## 추가 보너스

### 1) 속도 보너스

```
timeBonus = max(0, (기준시간 - 클리어시간) * 2)
```

---

### 2) 콤보 보너스

```
comboBonus = maxCombo * 5
```

---

### 3) 완벽 플레이 보너스

- 모든 단어 빠짐없이 빠르게 처리 시

```
perfectBonus = 50
```

---

## 최종 점수

```
totalPoint = 100 + timeBonus + comboBonus + perfectBonus
```

---

👉 평균 기대값: **120 ~ 250점**
