# 📦 하루상자 (HaruBox) 기획 개요

## 1. 서비스 컨셉
하루상자는 짧은 시간 안에 가볍게 즐길 수 있는 **아기자기한 놀이·게임·기록 콘텐츠를 모아둔 웹 플랫폼**입니다.

사용자는 하루상자 안에서:
- 게임을 즐기고
- 소소한 기록을 남기고
- 결과물을 저장하거나 공유하며
하루의 작은 재미를 수집합니다.

---

## 2. 핵심 방향성
- **인형뽑기(구 뽑기중독)**는 메인이 아닌 **대표 콘텐츠 중 하나**
- 로그인은 **선택적**
- 로그인 없이도 대부분의 콘텐츠 체험 가능
- 로그인 시 **기록·저장·개인화** 제공

---

## 3. 로그인 설계

### 3.1 로그인 원칙
- **초기 진입 허들은 낮게**
- "로그인 안 해도 됨" (Guest Play First)
- 단, 저장/히스토리/랭킹 등록은 로그인 필요

### 3.2 로그인 방식
- 소셜 로그인 위주 (Google, Kakao)
- 이메일/비밀번호 방식은 선택사항

### 3.3 로그인 시 제공 기능
- 플레이 기록 저장
- 개인 보관함 (이미지, 결과물)
- 콘텐츠별 통계
- 기기 변경 시 데이터 유지

---

## 4. 전체 페이지 구성

### 4.1 메인 페이지 (`/`)
**목적:** 하루상자의 분위기 전달 및 콘텐츠 탐색

*   **상단**:
    *   로고: 하루상자
    *   메뉴: 콘텐츠 | 로그인
*   **히어로 섹션**:
    *   문구: "오늘의 작은 재미를 담다"
*   **콘텐츠 카드 그리드**:
    *   인형뽑기 (Claw Machine)
    *   동물 운명연구소 (Pet Destiny Lab) - **NEW!**
    *   미니게임 (Coming Soon)
*   **하단**:
    *   서비스 설명
    *   SEO 텍스트

### 4.2 콘텐츠 목록 (`/contents` - 예정)
**목적:** 제공 중인 모든 놀이 콘텐츠 모음 (현재 메인 페이지에서 바로 접근 가능하도록 구현)

*   카드형 UI
*   상태 표시: NEW, 인기, 준비중
*   클릭 시 각 콘텐츠 상세/플레이 페이지로 이동

### 4.3 인형뽑기 (`/claw`)
*   **URL**: `/claw` (이전 `/` 및 `/game` 기능 통합)
*   **기능**:
    *   3D 인형뽑기 게임 (Game Logic Refer: `02_content_claw_machine.md`)
    *   물리엔진 기반 조작
    *   결과 이미지 저장 가능 (로그인 시)
    *   랭킹 시스템

### 4.4 동물 운명연구소 (`/pet-destiny`)
*   **URL**: `/pet-destiny`
*   **컨셉**: 반려동물과 집사의 생년월일을 기반으로 한 오행 명리학 분석 서비스
*   **주요 기능**:
    *   **성격 정밀 분석**: 타고난 기질(목, 화, 토, 금, 수), 생활 습관, 집사와의 관계 패턴 분석
    *   **건강 체크**: 오행 기반 취약 장기 및 건강 관리 팁 제공
    *   **운세 흐름**:
        *   평생 운세 (나이대별 흐름: 유년기, 성장기, 전성기, 노년기)
        *   올해의 운세 (종합, 애정, 건강, 재물)
    *   **궁합 & 마음**: 집사와의 궁합 점수(상생/상극) 및 마음의 문(신뢰도) 분석
    *   **캐릭터 카드**: 5가지 오행 속성을 반영한 귀여운 캐릭터 결과 카드 제공
    *   **교차 홍보 (Cross Promotion)**: 결과 페이지 하단에 고양이 건강 관리 앱 '묘록' 추천 배너 노출
        *   **위치**: 결과 콘텐츠 하단, 푸터 상단
        *   **구성**: 앱 아이콘 + 소개 텍스트("고양이 건강 기록 앱, 묘록") + CTA("앱 보러가기")
        *   **목적**: 자연스러운 서비스 추천 및 유입 유도

### 4.5 마이페이지 (`/mypage`)
*   로그인 정보
*   콘텐츠 이용 통계
*   데이터 초기화
*   탈퇴
