## AI 작업 지침
### 목적
치코런 서비스의 문제 생성 및 난이도 시스템을 1~100 레벨 체계와 초급/중급/고급 난이도 버튼에 연동되도록 관리한다.

### 작업 단계
1. **데이터 스키마 준수**: `Problem` 인터페이스에 정의된 모든 필드(`passage`, `difficulty`, `wordCount`, `questionType` 등)를 충실히 구현한다.
2. **난이도별 특징 반영**: 
   - 초급: 짧은 문장, 기본 문법.
   - 중급: 중간 길이, 추론/논리.
   - 고급: 장문(200~450단어), 수능형 유형 및 주제.
3. **UI 연동**:
   - 지문 영역 스크롤 및 크기 조정 (최대 65% 차지).
   - 단어 수 및 예상 시간 표시.
   - 난이도 아이콘(🔥, ⚡, 👑) 표시.
4. **Mock 데이터 생성**: `generateMockProblem` 함수를 통해 레벨과 난이도에 맞는 문제를 자동으로 생성할 수 있어야 한다.

### 주의사항
- 고급 레벨에서 쉬운 난이도를 선택하더라도 최소 Medium 이상의 문제가 나오도록 보정해야 한다.
- 해설은 MZ 스타일(1~2줄, 힙한 톤)을 유지하되 고급은 수능 스타일 멘트를 추가한다.
- 지문 내 키워드 볼드 처리는 'hard' 난이도에서만 선택적으로 적용한다.
