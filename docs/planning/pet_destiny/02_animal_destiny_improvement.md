# 동물 운명 콘텐츠 개선 기획 (Daily Animal Destiny)

## 1. 기획 목적
현재 동물 운명 콘텐츠가 가진 "짧고 단순하며 AI 느낌이 강하다"는 문제를 해결하고, 사용자에게 "읽는 재미"와 "신뢰감"을 주는 고품질 콘텐츠로 업그레이드한다.

- **문제점**: 짧은 텍스트, 부족한 깊이, 낮은 체류 시간
- **개선 목표**: 풍부한 설명 텍스트, 템플릿 조합 방식(AI 생성 지양), 정보량 강화

## 2. 전체 출력 구조
사용자에게 노출되는 결과는 아래와 같은 구조를 반드시 유지한다.

1. **[타이틀]**: "오늘 당신은 [동물명] 입니다 [이모지]"
2. **[긴 설명 텍스트]**: 3~5문단 분량의 서술형 분석
3. **[키워드 리스트]**: 고정된 5개의 핵심 태그 (#키워드1 #키워드2 ...)

## 3. 결과 구성 상세

### 3.1 설명 텍스트 구조 (3단 구조)
1. **오늘의 상태 해석**: 사용자의 오늘 컨디션, 분위기, 에너지 흐름 해석 (예: "오늘 당신은...", "현재 흐름은...")
2. **동물 성향 연결 분석**: 동물의 생물학적/상징적 특징과 사용자의 상태를 연결 (예: 행동 패턴, 대응 방식)
3. **오늘 하루 행동 가이드**: 실질적인 행동 조언 (하면 좋은 행동, 피하면 좋은 행동)

### 3.2 키워드 정책
- **개수**: 무조건 5개 고정
- **역할**: 설명을 요약하는 핵심 태그 (의미 중복 금지, 설명과 직접 연결)
- **노출**: 토글 없이 상시 노출

## 4. 기술적 구현 전략 (Template Engine)

### 4.1 AI 역할 최소화
- AI는 문장을 직접 생성하지 않고, **선별된 5개의 동물**과 **템플릿 조합**만을 결정한다.
- 모든 문장은 미리 작성된 고품질 템플릿 풀에서 선택된다.

### 4.2 데이터 구조 (Data Schema)
각 동물(Animal)은 아래의 데이터를 보유한다.
- `animalName`: 동물의 이름 및 이모지
- `introTemplates[]`: 오늘의 상태 해석 템플릿 풀
- `analysisTemplates[]`: 동물 성향 연결 템플릿 풀
- `guideTemplates[]`: 행동 가이드 템플릿 풀
- `keywords[]`: 해당 동물의 핵심 키워드 5개

### 4.3 문장 조합 규칙
- **조합 순서**: `intro` (1개) + `analysis` (1~2개) + `guide` (1~2개)
- **결과**: 긴 문단 형태의 텍스트 생성

## 5. 문체 가이드 (Tone & Manner)
- **지향**: 분석형 운세 느낌, 부드러운 대화체, 가볍지만 진지한 분위기
- **지양**: 지나치게 귀여운 말투, MBTI 스타일, SNS식 짧은 문장
- **목표**: "진짜 전문가가 나를 분석해준 느낌"

## 6. 최종 출력 예시
> **오늘 당신은 사막여우 입니다 🦊**
> 
> 오늘 당신은 평소보다 주변을 예민하게 관찰하고, 혼자만의 시간을 통해 에너지를 회복하려는 흐름이 강합니다. 외부 활동보다 내부 정리가 중요한 하루입니다.
> 
> 사막여우는 작은 변화에도 빠르게 반응하는 동물입니다. 지금의 당신 역시 사람보다는 상황을 읽는 능력이 더 예민하게 작동하고 있습니다.
> 
> 오늘은 무리하게 약속을 늘리기보다, 혼자 집중할 수 있는 일을 선택하는 것이 좋습니다. 특히 생각 정리나 계획 세우기에 좋은 날입니다.
